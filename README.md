# Tsunami
An advanced HTTP flooder written in Golang

__This project is huge WIP__

## Basic Usage
```bash
./tsunami http://example.com --threads 100
```

## Features
- Live attack stats
- Customizable mutlithreading
- HTTPS support __(Note: Certificates aren't verified)__
